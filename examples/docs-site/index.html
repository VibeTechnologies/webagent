Authentication

All API requests require a Bearer token in the Authorization header.

curl -H "Authorization: Bearer YOUR_TOKEN" https://api.example.com/v1/users

Endpoints

GET /v1/users

List all users. Supports pagination with page and limit query parameters.

POST /v1/users

Create a new user. Requires name and email in the request body.

{
  "name": "Jane Doe",
  "email": "jane@example.com"
}

GET /v1/users/:id

Get a single user by ID.

DELETE /v1/users/:id

Delete a user. Returns 204 No Content on success.
